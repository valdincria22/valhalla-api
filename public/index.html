API Docs
Referência completa das rotas disponíveis
GET/keys
Rota pública — retorna todas as keys ativas em texto puro (uma por linha). É essa URL que o script Lua usa.
https://valhalla-api.up.railway.app/keys
GET/validate?key=...&userid=...
Valida uma key. Vincula o userid ao primeiro uso. Retorna JSON com valid, plan, expires_at.
{ "valid": true, "plan": "Pro", "expires_at": "2026-12-31", "message": "OK" }
POST/generate
Gera uma nova key. Requer header x-admin-token.
curl -X POST https://valhalla-api.up.railway.app/generate \ -H "x-admin-token: SEU_TOKEN" \ -H "Content-Type: application/json" \ -d '{"plan":"Pro","note":"Cliente VIP"}'
POST/ban
Bane uma key imediatamente. Na próxima validação do script, o usuário é barrado.
curl -X POST https://valhalla-api.up.railway.app/ban \ -H "x-admin-token: SEU_TOKEN" \ -H "Content-Type: application/json" \ -d '{"key":"VLH-XXXX-XXXX-XXXX-0001"}'
POST/reset
Desvincula o userid de uma key, permitindo que outro usuário a use.
